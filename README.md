# Creative IT Park Website

A full-stack company website developed for Creative IT Park, designed to present company information, services, projects, and dynamic website content through a custom admin panel.

## Overview

This project was rebuilt from a previous Laravel-based website into a modern full-stack architecture using **React.js** for the frontend and **Node.js / Express.js** for the backend.

The system includes a public-facing company website and an admin panel that works as a content management system, allowing authorized users to manage website content without directly modifying the codebase.

## Key Features

- Modern company website built with React.js
- Backend API developed using Node.js and Express.js
- Custom admin panel for content management
- Dynamic content management for company pages and website sections
- Image and media content handling
- Responsive frontend layout
- Separation of frontend and backend codebase
- API-based communication between frontend and backend
- Maintainable project structure for future updates
