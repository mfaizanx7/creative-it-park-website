# Creative IT Park - Full-Stack Website

A modern, high-performance, full-stack web application rebuilt from a legacy Laravel codebase to a robust decoupled architecture using **React.js** for the frontend and **Node.js / Express.js** for the backend REST APIs.

## 🚀 Features
- **Modern Full-Stack Architecture:** Completely rebuilt from scratch for faster rendering and modular code maintenance.
- **Responsive User Interface:** Sleek, modern, and user-friendly pages designed to adapt to all screen sizes.
- **Robust Backend APIs:** Scalable RESTful API layers to handle client-side inquiries and website content dynamically.
- **Production Ready:** Optimized configurations prepared for live server deployments.

## 🛠️ Tech Stack
- **Frontend:** React.js, Redux, HTML5, CSS3, Bootstrap
- **Backend:** Node.js, Express.js, RESTful APIs
- **Database:** MongoDB / MySQL
- **Version Control:** Git & GitHub
